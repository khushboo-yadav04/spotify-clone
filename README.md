# 🎵 Spotify Clone (Frontend)

This is a **Spotify Clone UI project** built using HTML and CSS.
It is a beginner-friendly project created to practice frontend development skills.


## 🚀 Features

* 🎧 Spotify-like UI design
* 📁 Sidebar with Home & Library
* 🎵 Playlist cards layout
* 🎨 Clean and modern styling using CSS
* 💡 Responsive layout (basic)


## 🛠️ Tech Stack

* HTML5
* CSS3


## ⚠️ Note

This project is currently **incomplete** and under development.
More features like:

* Music player functionality
* JavaScript interactivity
* API integration

will be added in the future.
